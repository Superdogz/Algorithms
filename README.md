## How To Use
main.py has insertion sort, counting sort, and radix sort all in the same file.  
At the bottom, there are options to change the type of array (shuffled, reversed, custom).  
  
Sort the array by calling the specific sort function:  
&emsp; insertion_sort(unsorted_array)  
&emsp; counting_sort(unsorted_array)  
&emsp; radix_sort(unsorted_array)
